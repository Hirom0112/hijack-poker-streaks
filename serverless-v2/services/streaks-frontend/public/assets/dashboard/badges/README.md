# Streak milestone badges

Drop the 12 final badge PNGs here. Naming is keyed by **axis + milestone day**, so it
maps 1:1 to the milestone ladder the backend derives badges from (3/7/14/30/60/90).

- Format: PNG, transparent background, ~256×256, single medallion per file.
- Earned vs locked is rendered in CSS (grayscale/opacity) — ship ONE art file per badge.
- Style reference: `design-assets/dashboard/badges/REFERENCE-trophy-shelf.png` (gitignored).
